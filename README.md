## KhumaloCraft Emporium

**Project Overview**
KhumaloCraft Emporium is a cloud-native e-commerce platform deployed on Azure. It connects artisans to customers worldwide, providing a scalable and secure online storefront.

**Problem & Solution**

* **Problem:** The client needed a global, scalable platform but lacked technical infrastructure for a reliable e-commerce solution.
* **Solution:** Full-stack web app deployed on Azure with product catalogues, order processing, and advanced cloud services for search and automation.

**Key Features / Functionalities**

* Azure App Service deployment with Azure SQL Database
* Full product catalogue with images, pricing, and descriptions
* Admin interface for inventory and order management
* Azure Cognitive Search for advanced product search
* Azure Durable Functions for automating orders, inventory updates, and notifications
* Relational database design for users, products, and transactions

**Tech Stack**
ASP.NET Core MVC, Azure App Service, Azure SQL, Azure Cognitive Search, Azure Durable Functions, Entity Framework Core

**Setup Instructions**

1. Clone the repo: `git clone https://github.com/SandeepHarii/KhumaloCraft-Emporium`
2. Open in Visual Studio
3. Update Azure SQL Database connection string in `appsettings.json`
4. Deploy to Azure App Service or run locally
